File Permissions in Linux
The file permissions are categorized into 3 ways;
users (u)		- read (r), write (w), execute (x)
group (g)	- read (r), write (w), execute (x)
others (o)	- read (r), write (w), execute (x)

rw-r--r--		kiran.txt

For users ----> read and write
For group ----> only read
For others ----> Only read

default permissions for a file			- rw-r--r--
default permissions for a directory		- rwxr-xr-x

To edit the file permissions we will use "chmod" command
chmod u+x kiran.txt
chmod g+wx kiran.txt
chmod o+wx kiran.txt
chmod u-x kiran.txt
chmod g-rwx kiran.txt
chmod o-rx kiran.txt

File permissions using numbers
0 - no permissions
1 - execute
2 - write
3 - 2 + 1 - write + execute
4 - read
5 - 4 + 1 - read + execute
6 - 4 + 2 - read + write
7 - 6 + 1 - read + write + execute
